# Electronics Store Web Project

This is a responsive *ReactJS* website for selling electronics products. It has a clean, professional design and supports both *light mode* and *dark mode. The website uses **Bootstrap* for styling and works on both desktop and mobile devices.

## Pages

The website includes the following pages:

1. *Home* – Main landing page with featured products.
2. *Products* – Dynamic page displaying the electronics products available for sale.
3. *Contact* – Page with a contact form to reach out to the store.
4. *Cart* – Page to view, update, or remove items added to the shopping cart.

## Features

- Responsive design (desktop and mobile friendly)
- Light and dark mode toggle
- Add, update, and remove products from the cart
- Clean UI with Bootstrap styling
